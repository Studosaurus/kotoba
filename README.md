# Kotoba

Kotoba is a modular AI-powered Japanese learning platform.

The project source of truth lives in:

- `VISION.md`
- `ARCHITECTURE.md`
- `ROADMAP.md`
- `DECISIONS.md`

This repository currently contains the application scaffold only. Feature business logic is intentionally deferred.

## Stack

- Next.js App Router
- TypeScript
- React
- Tailwind CSS
- Supabase and PostgreSQL
- Supabase Auth
- Vercel
- Progressive Web App

## Development

Copy `.env.example` to `.env.local` and provide local Supabase values before running the app.

```bash
npm install
npm run dev
```
